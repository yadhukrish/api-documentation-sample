# OpenWeather Current Weather API — Documentation Sample

A technical writing portfolio sample demonstrating RESTful API documentation, structured reference writing, and docs-as-code practices.

---

## About This Sample

This repository documents the [OpenWeather Current Weather API](https://openweathermap.org/current). It is written for developers who need to retrieve real-time weather data by city name, ZIP code, or geographic coordinates.

**Documentation type:** API Reference + Quickstart  
**Audience:** Developers integrating weather data into applications  
**Writing style:** Google Developer Documentation Style Guide  
**Tooling:** Markdown, GitHub, Postman, cURL
